<h3> 🚀 Question 1: Basics of Function Definition</h3>

**Description🔍**: Write a function named greet() that prints "Hello, World!". Then, call the function.

---

## Code📝
```python
def greet():
    print("Hello, World!")

# Calling the function
greet()

```
---

## Example Output📊  
**Output:** ```Hello, World!```
